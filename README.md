# PotatOS MC Server Website

基于 [Astro](https://astro.build) + [Starlight](https://starlight.astro.build) 构建的 Minecraft 服务器官方网站。  
为 **PotatOS** 服务器提供文档、公告、玩法介绍、成员指南等模块化内容展示。

## ✨ 特性

- 🧱 使用 Starlight 主题，开箱即用的文档结构
- ⚡ Astro 静态站点生成，性能极佳
- 🎨 支持暗色 / 亮色主题切换
- 📱 响应式设计，移动端友好
- 🧩 易于自定义组件和布局


## 🚀 快速开始

>[!Note]
>首先需要安装：
>- Node.js
>- pnpm

1. 将此仓库克隆至本地
   `git clone https://github.com/momo-OwO-qwq/PotatOS.git`
2. 移动至仓库根目录
   `cd PotatOS`
3. 安装依赖
   `pnpm install`
4. 构建文档
   `pnpm build`
5. 本地预览
   `pnpm dev`
   

## 🙌 贡献

1. Fork 此仓库至自己的 GitHub 账户下
2. 将自己账户下的仓库克隆至本地
   `git clone https://github.com/你的用户名/PotatOS.git`
3. 在本地仓库中进行修改并构建文档测试，无误后 push 至自己的仓库
4. 回到此仓库，点击 Pull requests -> New pull request 发起 PR
